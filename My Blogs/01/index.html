How to lock any file on PC/Mac

In this blog, I will tell you how you can lock any type of file in your PC or Mac using a simple method. I will be using WinRar here which is a very popular application and most of the computers will be having it.


Now let us move on to the steps:


Step 1:
First, download and Install WinRar if you haven't.


Step 2:
Locate the file which you want to lock. Right click on it and click on "Add to archive..."



Step 3:
Click on the "Set password" button on the window that has opened. Set the password and click "OK". And again click on "OK".





Now you are done! A new file with '.rar' extension will be created. If you try to open it, by double clicking on it, it will ask for the password.


Step 4:
The last step that you are to do, is to delete the original file.
